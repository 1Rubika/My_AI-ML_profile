# 🤖 My_AI-ML_Profile

A modern AI & Machine Learning portfolio built with React, TypeScript, Vite, and Tailwind CSS to showcase projects, skills, certifications, achievements, and professional experience.

## ✨ Features

- 🎯 Responsive and modern UI
- 👤 About Me section
- 💻 AI & ML Projects
- 🏆 Achievements & Certifications
- 🎓 Education
- 📚 Technical Skills
- 🌐 Social & Professional Links
- 📬 Contact Section
- 📱 Mobile-friendly design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/1Rubika/My_AI-ML_Profile.git
```

### Navigate into the project

```bash
cd My_AI-ML_Profile
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

---

## 📦 Production Build

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
